Questionario Formulario

a) Na tag < input > com atributo type="text", para que serve o atributo "placeholder"?

O atributo "placeholder" é usado para fornecer um texto de exemplo dentro do campo de entrada (input), indicando aos usuários qual tipo de informação deve ser inserida. Por exemplo, um campo de entrada para o nome pode ter um placeholder como "Digite seu nome".

b) Na tag < input > com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?

O atributo "checked" é usado para pré-selecionar um botão de rádio (radio button) ou uma caixa de seleção (checkbox). Quando este atributo está presente, o botão de rádio ou a caixa de seleção é exibido marcado por padrão.

c) Na tag < input > para que serve o atributo "value"?

O atributo "value" é usado para definir o valor inicial de um campo de entrada (input). Quando o formulário é enviado, o valor atribuído a esse atributo é enviado junto com os outros dados do formulário.

d) Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

O atributo "name" é utilizado para agrupar vários botões de rádio em um formulário, garantindo que apenas um deles possa ser selecionado de cada vez dentro desse grupo.

e) Para que serve a tag < label >? Para que serve o atributo "for" nessa tag?

A tag < label > é usada para rotular elementos de formulário, fornecendo uma descrição ou rótulo para eles. O atributo "for" especifica qual elemento de formulário está associado ao rótulo. Ao clicar no rótulo, o navegador focaliza automaticamente o elemento associado

f) Na tag < option > para que serve o atributo "selected"?

O atributo "selected" é usado para especificar que uma opção em um menu suspenso (select) está inicialmente selecionada quando a página é carregada.

g) Nas tags de formulário para que serve o atributo "required"?

O atributo "required" é usado para indicar que um campo de entrada é obrigatório e não pode ser enviado vazio. O navegador impedirá o envio do formulário até que todos os campos obrigatórios sejam preenchidos.

h) Para que serve a tag < textarea >?

A tag < textarea > é usada para criar uma área de texto multi-linha em um formulário, permitindo que os usuários insiram texto mais longo, como comentários ou mensagens.

i) Na tag < form > para que serve o atributo "action"?

O atributo "action" especifica para onde os dados do formulário serão enviados quando o formulário for submetido. Pode ser uma URL para um script de processamento do lado do servidor ou outra página.

j) Na tag < form > para que serve o atributo "method"?

O atributo "method" especifica o método HTTP a ser usado ao enviar os dados do formulário. Os valores comuns para esse atributo são "GET" e "POST".

k) Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag < form >?

Para garantir que o valor digitado de uma senha não apareça na barra de endereços do navegador, você deve usar o método "POST" no atributo "method" da tag < form >. Isso envia os dados do formulário no corpo da solicitação HTTP, em vez de como parte da URL.

l) Qual valor deve ser informado no atributo "method" na tag < form > em um formulário que será usado para fazer upload de arquivos?

Para um formulário que será usado para fazer upload de arquivos, você também deve usar o método "POST" no atributo "method" da tag < form >, pois os dados de arquivo não podem ser incluídos na URL de uma solicitação "GET".

m) O que as tags < fieldset > e < legend > produzem?

A tag < fieldset > é usada para agrupar elementos relacionados em um formulário, enquanto a tag < legend > fornece uma legenda ou título para o < fieldset >, descrevendo o propósito ou contexto do grupo de elementos. Isso pode ser útil para organizar visualmente e semânticamente partes do formulário.
